Curso Análise Gráfica no R

Bem-vindo! Neste curso você vai aprender a manipular bancos de dados e gerar incríveis gráficos 📊📈 oferecidos pela linguagem R.

Esse curso foi desenvolvido pelo prof. Weverton Costa para o curso Análise Gráfica no R oferecido pela Conecta GEM.

Descrição

Este curso possibilita que o aluno seja independente nas construções gráficas utilizando o R studio. Com uma base prática, voltada para desenvolver habilidades e criatividade durante o processo de construção de gráficos. As aulas serão gravadas e disponibilizadas durante 30 dias para todos os participantes que desejam rever o conteúdo, ou não consigam acompanhar na data estabelecida do curso.

Pré-requisitos

  • Para o melhor aproveitamento do curso, é desejável conhecimentos básicos de programação em R ou ser curioso em aprender.

  • Software R e RStudio com versão atual ou recentes.

Para quem é este curso:

Para todos aqueles que desejam aprender a produzir gráficos com execelência.

O que você irá encontrar nesse curso:

  • Tutorial: Tudo que você precisa para usar este material e como realizar as instalações dos softwares, pacotes e materiais auxiliares para você se aprofundar mais sobre o R e análise de dados.

  • Introdução ao R: Para que você possa ter mais conhecimento sobre o R, preparei um material de introdução ao R. Assim você pode aprender mais sobre o software e como utilizá-lo, além de e poder consultar futuramente em seus estudos.

  • Manipulação de dados: Aqui você irá aprender sobre manipulação de dados. A análise gráfica depende desse conhecimento para futuros ajustes ou até mesmo para a produção dos gráficos da maneira que você desejar. Neste tópico iremos abordar as principais funções do framework tidyverse, que é amplamente utilizado para manipulação de dados, permitindo realizar manipulações de maneira rápida e simples e organizada.

  • Análise Gráfica no R: O objetivo deste curso é te ensinar todos os conceitos para fazer gráficos usando a biblioteca ggplot2. O ggplot2 é uma biblioteca que permite construir vários tipos de gráficos diferentes, com estruturas e estética diferenciadas e de qualidade.
